# AquaFish

[![Build Status](https://travis-ci.org/mrsndmn/AquaFish.svg?branch=master)](https://travis-ci.org/MrSndmn/AquaFish)

Student project. Aquarium for your fish :)

Control:
    [SPACE]     -- create new fish
    [R]         -- delete fish
    [LSHIFT+R]  -- delete all fishes
